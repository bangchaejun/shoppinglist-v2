# ShoppingList V2

기존 로컬 기반 쇼핑리스트 앱을 참고하여, 실제 서비스형 웹앱으로 재구축하는 프로젝트입니다.

## 목표
- 사용자별 쇼핑리스트 관리
- 리스트별 아이템 및 수량 관리
- 예산 설정 및 잔액 계산
- 구매 이력 저장
- 모바일 친화적 UX 제공

## 1차 MVP
- 회원가입 / 로그인
- 쇼핑리스트 생성 / 조회 / 삭제
- 아이템 추가 / 수정 / 삭제
- 수량 조절
- 구매완료 처리
- 예산 / 지출 / 잔액 관리
- 구매 히스토리

## 기술 스택
- Next.js
- TypeScript
- Supabase
- Vercel

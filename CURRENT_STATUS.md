# CURRENT_STATUS

## 프로젝트 개요
- 프로젝트명: shoppinglist-v2
- 기술스택: Next.js, TypeScript, Supabase
- 목적:
  집에서 장보기 예산과 품목을 계획하고,
  마트에서 실제 수량과 가격을 입력하면서
  카트 총액과 남은 예산을 실시간으로 확인하는 앱

## 현재 완료된 것
- Next.js 프로젝트 생성
- GitHub 저장소 연결 완료
- Supabase 프로젝트 생성 및 연결 완료
- 회원가입 가능
- 로그인 가능
- 이메일 인증 없이 로그인 가능하도록 우회 처리 완료
- 대시보드에서 쇼핑리스트 생성 가능
- shopping_lists 테이블 생성 완료
- shopping_items 테이블 생성 및 구조 확장 완료
- 리스트 상세 페이지에서 항목 추가 가능
- planned_quantity / actual_quantity / unit_price / total_price / is_in_cart 구조 반영
- 카트 반영 시 현재 카트 총액 / 남은 예산 실시간 표시 가능

## 현재 핵심 기획
- 집에서 예산과 필요한 품목만 미리 저장
- 마트에서 리스트를 보며 실제 수량과 가격 입력
- 앱에서 카트 담기 처리
- 현재 카트 총액과 남은 예산을 실시간 비교
- 마트에서 현장 추가 항목도 바로 입력 가능
